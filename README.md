# 💰 Testes de QA - Sistema de Descontos

Projeto de testes manuais focado na validação de regras de negócio para cálculo de descontos.

---

## 🎯 Objetivo
Garantir que o sistema:
- Aplique corretamente os descontos
- Não acumule descontos indevidos
- Retorne valores corretos ao usuário

---

## 🧪 Tipos de Teste
- Testes Funcionais
- Testes de Regras de Negócio
- Testes Exploratórios

---

## 📂 Estrutura do Projeto

| Arquivo | Descrição |
|--------|--------|
| test-plan.md | Planejamento dos testes |
| test-cases.md | Casos de teste |
| bugs.md | Registro de falhas encontradas |

---

## 🚀 Cenários Testados
- Aplicação de desconto único
- Aplicação de múltiplos descontos
- Validação de valores inválidos
